### Installation

#### Use localhost

```
pnpm install
```

#### Use devcontainer in vscode

install vscode extension [**Remote - Containers**](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

### Started

```
pnpm dev
```
